# Aiva Icon Design Spec

**Date**: 2026-05-03
**Status**: Approved

## Overview

为 Aiva macOS 桌面应用设计两套图标：Dock 图标（App Icon）和菜单栏 Tray 图标。

设计方向：Siri 式光球风格，紫蓝青渐变，传达 AI 助手的智能存在感。

---

## 一、Dock 图标（App Icon）

### 外形

macOS 标准圆角矩形（squircle），深色背景内嵌光球。

### 尺寸

- Master: 1024x1024 PNG
- 由 electron-builder 自动生成 `.icns`（含 16/32/64/128/256/512 各尺寸）
- 存放路径: `resources/icon.png`

### 结构（由外到内）

| 层级 | 规格 | 说明 |
|------|------|------|
| 外框 | squircle，圆角约 22% | macOS 标准 app 图标形状 |
| 背景 | `#0f0a1e` 深色填充 | 深蓝黑底色 |
| 光球 | 径向渐变，r 约占框体 65% | 中心 → 外缘渐变 |
| 核心高光 | 偏上方，r 约占光球 65% | 白色到淡紫 |
| 反光条 | 顶部椭圆 | 微弱环境光反射 |
| 边框 | 1px 白色，alpha 0.06 | 极细半透明描边 |

### 光球渐变参数

径向渐变（Radial Gradient，中心向外）：

| Offset | 颜色 | Alpha |
|--------|------|-------|
| 0% | `#ff6b9d`（粉） | 0.95 |
| 20% | `#c44dff`（紫） | 0.80 |
| 50% | `#6e8efb`（蓝） | 0.50 |
| 80% | `#1e1b4b`（深蓝紫） | 0.30 |
| 100% | `#0f0a1e`（背景色） | 0.00 |

### 核心高光参数

径向渐变（偏上方，cy 约 42%）：

| Offset | 颜色 | Alpha |
|--------|------|-------|
| 0% | `#ffffff` | 0.70 |
| 35% | `#f0d0ff` | 0.35 |
| 100% | `#c44dff` | 0.00 |

### 反光条

- 形状：椭圆，rx 约为光球半径的 38%，ry 约为 18%
- 颜色：白色
- Alpha：0.08

### 边框

- 宽度：1px（相对 1024 master）
- 颜色：白色
- Alpha：0.06

## 二、菜单栏 Tray 图标

### 设计理念

Dock 光球的"压缩版"——用 4 层不同透明度的同心圆模拟径向渐变效果。

### 尺寸

- 44x44 像素（Retina 下显示为 22pt）
- 使用 macOS template 图标规范（单色 + alpha 通道）

### 主体结构

4 层同心圆，由外到内 alpha 递增：

| 层级 | 半径（64x64 坐标系） | Alpha（深色模式白色） | Alpha（浅色模式黑色） |
|------|---------------------|----------------------|----------------------|
| 外层 | 28 | 0.10 | 0.06 |
| 中外 | 20 | 0.18 | 0.12 |
| 中内 | 12 | 0.35 | 0.22 |
| 核心 | 4 | 1.00 | 0.85 |

### 状态指示点

- 位置：右上角
- 尺寸：约 6px（显示尺寸），带 1.5px 描边（匹配背景色）
- 颜色对应：

| 颜色 | 色值 | 状态 | 含义 |
|------|------|------|------|
| 灰色 | `#8e8e93` | idle | 待命 |
| 蓝色 | `#32adff` | recording/transcribing | 录音/转写中 |
| 绿色 | `#34c759` | executing | 执行中 |
| 红色 | `#ff453a` | error | 错误 |
| 黄色 | `#ffd60a` | editing | 编辑中 |

### 实现方式

- 代码生成（`electron/tray.ts`），使用 RGBA buffer 像素级绘制
- 基础图标（baseIcon）不含状态点，使用 macOS template 图标（`setTemplateImage(true)`）
- 状态点图标（dotIcons）在基础图标上叠加彩色小点
- 继续使用现有的 `NativeImage.createFromBuffer` 方式

## 四、设计决策记录

| 决策 | 选择 | 原因 |
|------|------|------|
| 风格 | 光球风（Siri 式） | 用户偏好 AI 助手存在感 |
| 配色 | 紫蓝青渐变 | 经典 AI 配色，视觉冲击力强 |
| Dock 形状 | macOS 标准 squircle | 符合 HIG 规范 |
| 光球方案 | 纯净光球（无额外元素） | 各尺寸清晰可辨 |
| Tray 设计 | 4 层渐变圆盘 | 光球压缩版，干净有层次 |
| 状态点位置 | 右上角 | 用户偏好，比右下角更自然 |
